# 株式会社アドバンス - コーポレートサイト

中小企業のビジネス成長を支える、信頼のデジタルパートナー

## 🎯 プロジェクト概要

株式会社アドバンスの公式コーポレートサイトです。システム開発、ホームページ制作、WEBマーケティング、AI研修の4つの事業領域で、中小企業のデジタル化を総合的にサポートします。

## 🚀 使用方法

### ローカル環境での表示

1. **ファイルをダウンロード**
   ```bash
   # プロジェクトをダウンロード
   ```

2. **ブラウザで開く**
   - `index.html` をダブルクリックしてブラウザで開く

3. **ローカルサーバーでの起動（推奨）**
   ```bash
   # Python 3の場合
   python -m http.server 8000
   
   # Node.jsのhttp-serverを使う場合
   npx http-server
   ```
   
   その後、`http://localhost:8000` でアクセス

## 🔧 カスタマイズ方法

### 画像の変更
HTMLファイル内の画像URLを変更してください：
```html
<img src="your-image-url.jpg" alt="説明文">
```

### カラーの変更
`css/style.css` の `:root` セクションで色を変更できます：
```css
:root {
    --color-primary: #2563EB;    /* メインカラー */
    --color-accent: #10B981;     /* アクセントカラー */
}
```

### コンテンツの変更
各HTMLファイル内のテキストを直接編集してください。
