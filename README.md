## 🌐 デプロイ方法

詳細な設置手順は `INSTALLATION.md` をご参照ください。

### 簡易手順
1. 全ファイルをレンタルサーバーの公開ディレクトリにアップロード
2. パーミッション設定（HTML/CSS/JS: 644、ディレクトリ: 755）
3. ブラウザでアクセスして動作確認
4. SSL証明書の設定（推奨）

### 推奨サーバー要件
- Webサーバー: Apache 2.4以上 / Nginx 1.18以上
- PHP: 不要
- データベース: 不要
- ディスク容量: 10MB以上
