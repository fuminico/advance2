## 🚀 使用方法

### ブラウザで直接表示
1. `index.html` をダブルクリック
2. すべてのページが正常に表示されます
3. ナビゲーションリンクから各ページに移動可能

### ローカルサーバーで起動（推奨）
```bash
# Python 3の場合
python -m http.server 8000

# Node.jsの場合
npx http-server
```
その後、http://localhost:8000 でアクセス
