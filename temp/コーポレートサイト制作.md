# RAG情報: コーポレートサイト制作

コーポレートサイト制作**サービスは、単なる制作にとどまらず、**ターゲットに応じたコンテンツと導線を整理し、目的を達成する制作**を特徴としています。これは、キーワード分析、内部対策、コンテンツSEOまでを設計に含めた**ビジネス課題を解決するコンサル型制作**として位置づけられています。

## 1. コーポレートサイトの定義と重要性

コーポレートサイトは、その名の通り**企業情報を伝えるためのホームページ（WEBサイト）**です。企業の顔として、**信頼性やブランディングが重要視**されます。

コーポレートサイトは、自社サービスの顧客、取引先企業、採用候補者、投資家など、**さまざまな目的の異なる方々**に対して、サービス情報、企業情報、採用情報、IR情報といった**幅広い情報がアップされる**ことが多い媒体です。すべての企業がコーポレートサイトを持つのが基本となる中で、いかに正確に必要な情報を伝え、どのように伝えていくかが、**ビジネスの結果をも左右する**ため重要性が増しています。

アドバンスは、デジタルマーティングを包括的に扱っている企業として、コーポレートサイトに関する課題を**マーケティング視点**で解決します。

## 2. コーポレートサイト制作における一般的な課題

デジタルマーケティングの視点から見た場合、コーポレートサイト制作/リニューアルにあたっては、以下のような課題が頻繁に見られます。

*   **情報設計と導線設計の課題**: サービスコンテンツ、企業情報コンテンツ、採用コンテンツなどの情報設計の優先度と導線設計ができておらず、来訪者がホームページ（WEBサイト）で迷い、離脱してしまう。
*   **情報不足**: コーポレートサイト1サイトのみの運営で、自社サービスの購入までに必要な情報が掲載されていないという改善依頼が営業サイドから来ている。
*   **ブランドとの乖離**: 自社として打ち出すべきビジョン、ミッションやブランドイメージと現在のホームページ（WEBサイト）が乖離している。
*   **分析不足**: ホームページ（WEBサイト）の数字分析ができておらず、現状の課題を理解できていない、または現状の改善イメージがない。

## 3. コーポレートサイト制作の5つのポイントと戦略

コーポレートサイトが貴社にとってどのような役割を持たせているかを中心に戦略を練ることが基本となります。制作を成功させるために、アドバンスは以下の5つのポイントを提示しています。

### ポイント1: 現状ホームページ（WEBサイト新設）における課題の確認
*   **リニューアルの場合**: 現状をヒアリングし、アクセス状況などの数値分析を行った上で、課題を抽出、解決策を検討し、KGI/KPIなどを設定します。
*   **新設の場合**: コーポレートサイト自体の目的を整理し、サイトを作る上での課題を確認します。

### ポイント2: コーポレートサイトの目的の設定

コーポレートサイトは複数の役割、目的を兼務していることが多いです。企業にとっての重要性や予算感によって、サービスサイトや採用サイトは目的別に別ホームページ（WEBサイト）化することも検討されます。まずは貴社のコーポレートサイトとして何を目的とするかを明確にする必要があります。

| コーポレートサイトの目的例 |
| :--- |
| **企業ブランディング** |
| **商品・サービス認知、販売** |
| **アフターサポート** |
| **採用活動** |
| **IR** |

### ポイント3: 目的別ターゲットユーザーの設定

複数の目的に渡るため、各目的別にどのようなターゲットが想定されるかを設定します。これは、ターゲットユーザーに合致したデザインの考察や、必要なコンテンツを探る上での土台となります。

| ターゲットユーザーの設定例 |
| :--- |
| **【サービス顧客】**：関東圏在住の20－30台の男性で、初訪での流入者が中心。 |
| **【パートナー候補企業】**：商品の販売を販売代理店候補でBtoC商材の販売に知見がある企業。 |
| **【採用候補者】**：中途採用中心で技術職は同業経験者または関連業界からの応募者、営業職は法人営業またはサービス業経験者。 |

### ポイント4: 目的別のコンテンツの用意と優先順位付けによる導線整理

各目的別にターゲットの重要度と優先順位を決め、コーポレートサイト全体の設計を行います。ターゲットに必要なコンテンツを配置し、どのようにカテゴライズし、どのようにレイアウトしていくかを決定します。特にトップページのレイアウトやナビゲーション、各種目的別コンテンツ間の遷移に注意し、**導線上で混線が起きないように**します。

**アドバンスサイト事例（ターゲット区分と情報優先順位）**（図解情報を含む）:

1.  **サービス顧客（メインはサービス情報、サブとして企業情報）**：**最重要ターゲット**。顧客ニーズから必要とされるサービス情報をメインとし、コンテンツを配置。
2.  **採用候補者（メインは採用情報。ナビゲーションから採用サイトへ遷移）**：採用関連媒体からの流入の比率が高いため、メインコンテンツを阻害せず、訴求力を下げないように下層に専用サイトを設置。検索エンジンからも採用サイトへダイレクトに遷移する候補者も想定。
3.  **パートナー企業導線**：重要度は上記より下がるため、ページ最下部に設置。

### ポイント5: 場合によっては目的別にホームページ（WEBサイト）を分離したほうがいい場合も

大抵の来訪者は上記のいずれかの目的をもってコーポレートサイトを訪問するため、**目的に特化したコーポレートサイトであるほど一般的に成果は高くなります**。ただし、別ホームページ（WEBサイト）にすることで、各サイトのコンテンツボリュームが薄くなり、検索エンジンからの流入量に影響を与えるため、**集客の観点から考慮が必要**です。

## 4. アドバンスの提供体制

アドバンスは、デジタルマーケティング全般を扱い、コーポレートサイト制作において**如何に機能をさせるか**を基準とした制作を行っています。貴社商品、サービスに加えて、貴社の営業フロー、マーケティングフロー、コーポレートサイトでの**全目的とターゲットの明確化が不可欠**であると考えています。

提供されるサービスは、「デジタルマーケティング（集客・改善コンサルティング）」と「WEBサイト制作（WEBサイト制作・運用）」の二本柱です。

アドバンスのサービスラインナップには、コーポレートサイト制作の他にも、ホームページ制作、採用サイト制作、サービスサイト制作、LP制作、オウンドメディア制作といったWEBサイト制作サービスや、WEB広告運用代行（リスティング広告、SNS広告など）、SEOコンサルティングなどが含まれています。
