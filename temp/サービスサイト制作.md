# サービスサイト制作（株式会社アドバンス）

RAG情報ファイル

## 1. サービス概要

株式会社アドバンスが提供するサービスサイト制作は、
**流入増加・コンバージョン増加を目的とした「成果基準のコンサル型WEBサイト制作」** を特徴とする。

単なる制作ではなく、以下を包括的に行う点が特徴：

* ターゲット区分による訴求設計
* CVユーザー分析に基づくコンテンツ整備
* キーワード分析 / 内部対策 / コンテンツSEO設計

---

## 2. サービスサイトの定義と役割

### 2-1. 定義

サービスサイトとは、提供するサービス・商品の理解を深め、**購入までをサポートするWEBサイト**。

### 2-2. 役割と目的

* **PRに特化**したページ構成により、商材の詳細理解を深める
* コーポレートサイトとは目的が異なり、**特定サービスの成約向上に最適化**
* 営業活動・マーケティング活動と連動し、課題解決を担う

---

## 3. サービスサイト制作における一般的な課題

アドバンスがよく遭遇する課題は以下の通り。

### 3-1. よくある課題

* **情報不足**

  * 購入に必要な情報が掲載されておらず、営業から改善依頼が来る
* **導線設計の不備**

  * 問い合わせや購入までの流れが最適化されていない
* **SEOによる集客不足**

  * 対策不足で必要キーワードで上位表示されていない
* **集客戦略の停滞**

  * 自社名・サービス系キーワード以外の流入が増やせない
* **分析不足**

  * 数字分析ができておらず、現状の課題が把握できていない

---

## 4. サービスサイト制作の5つのポイント（戦略）

### **ポイント1：現状課題の確認**

* **リニューアルの場合**

  * ヒアリング、アクセス解析、課題抽出、KGI/KPI設定
* **新規制作の場合**

  * サイト目的の整理、発生しうる課題の確認

### **ポイント2：ターゲットユーザーの設定**

ターゲットを明確化し、認知度・顕在/潜在割合・初訪/再訪比率などを判断。

#### 設定項目例

* ターゲットの認知度は？
* 顕在層と潜在層の割合は？
* 初訪と再訪の比率は？

**例：**
「関東圏在住の20～30代男性・初訪中心」の場合 →
基本情報を重視し、アフターサポート情報は導線の邪魔にならない位置に配置。

### **ポイント3：CVまでのカスタマージャーニー設計**

カスタマージャーニーマップを作成し、流入からCVまでの情報設計を行う。

#### ジャーニー例（フェーズ別）

| フェーズ  | 思考例            | 対応策例            |
| ----- | -------------- | --------------- |
| 興味・関心 | 成功方法を知りたい      | SEO強化・コンテンツ訴求   |
| 情報収集  | 課題の解決策を探したい    | リスティング広告、メディア掲載 |
| 比較検討  | どこに依頼すべきか比較したい | 比較系コンテンツの整備     |
| 購入    | 申し込みたい         | 申込導線の明示         |
| 購入後   | フォローがほしい       | サポート情報の強化       |

#### 掲載すべき主要コンテンツ

* サービス詳細
* 他社比較
* 導入事例
* 料金
* FAQ
* コラム
* セミナー
* 資料DL
* 会社情報

### **ポイント4：流入経路に応じたサイト設計**

流入元：自然検索 / WEB広告 / SNS / セミナー / メルマガ / テレアポ / 展示会 など
→ 各流入経路ごとに最適な導線とコンテンツを設計。

### **ポイント5：SEO/コンテンツマーケティングによる流入強化**

以下の2軸で強化：

#### 1. SEO基盤づくり（内部対策）

* 検索エンジンの上位表示ロジックに合わせた内部最適化

#### 2. 継続的流入強化（コンテンツマーケティング）

* 必要な情報コンテンツを継続追加
* 検索順位と流入増加を促進

**キーワード候補例**

* 指名キーワード：社名・商品名
* サービス関連キーワード
* 潜在層向けキーワード
