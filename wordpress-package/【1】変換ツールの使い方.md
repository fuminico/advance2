# 【1】変換ツールの使い方

**ファイル名:** `converter.html`

---

## 🚀 使用方法

### STEP 1: ツールを開く

`converter.html`をブラウザで開く（ダブルクリック）
- Chrome、Firefox、Edge推奨

---

### STEP 2: ファイルを選択

1. **HTMLファイルを選択**
   - 例: `original-html/index.html`
   - 「読込完了: index.html」と表示される

2. **CSSファイルを選択**
   - `css/style.css`
   - 「読込完了: style.css」と表示される

3. **JavaScriptファイルを選択**
   - `js/main.js`
   - 「読込完了: main.js」と表示される

---

### STEP 3: 子テーマ名を確認

- デフォルト: `advance-child-theme`
- 通常は変更不要

---

### STEP 4: 変換を実行

1. **「変換を実行」ボタンをクリック**
2. 画面下部に変換後のHTMLが表示される

---

### STEP 5: ダウンロード

1. **「ダウンロード」ボタンをクリック**
2. `index-converted.html`がダウンロードされる

---

### STEP 6: 次のファイルへ

1. **ページをリロード（F5キー）**
2. 次のHTMLファイルを選択
3. STEP 2〜5を繰り返し

**16ファイルすべて変換する所要時間:** 約8分（1ファイル30秒）

## 🐛 トラブルシューティング

### 問題: 変換ボタンが押せない

**原因:** 3つのファイルすべてが読み込まれていない

**解決策:** 各ファイルの下に「読込完了」と表示されているか確認

---

### 問題: ファイルが選択できない

**解決策:**
- ブラウザを最新版に更新
- 別のブラウザで試す（Chrome推奨）

---

## ✅ 変換内容

### 変換前
```html
<link rel="stylesheet" href="css/style.css">
<img src="images/hero.jpg">
<a href="service-system.html">
<script src="js/main.js"></script>
```

### 変換後
```html
<style>/* CSS全体 */</style>
<img src="/wp-content/themes/advance-child-theme/images/hero.jpg">
<a href="/service-system/">
<script>// JavaScript全体</script>
```
